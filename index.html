Real-Time Intelligent Road Safety Monitoring System

RoadSafety is an AI-driven ecosystem unifying four interconnected modules into a proactive transportation security framework. We monitor the overall traffic scenario and internal driver behavior using YOLOv11, EasyOCR, and hybrid CNN-LSTM architectures. Our solution features a robust Python core, a Flutter mobile application for driver notifications, and a React application for real-time authority monitoring.

Project Domain

Research Problem

Urban cities are experiencing a severe escalation of traffic infractions, severe collisions, and unpredictable security threats on the roads. The inherent inefficiency of manual surveillance systems fails to provide proactive and immediate responses. There is an urgent need for an automated, real-time framework to reliably address these compounding dangers and protect all road users.

1. Real-Time Violence Detection

Recognizes violent actions and weapon presence using a hybrid CNN-LSTM architecture combined with YOLOv11 for robust, high-speed spatial-temporal analysis.

2. Vehicle Violation Management

Deploys YOLOv11 and EasyOCR on CCTV feeds to dynamically detect helmetless riding, triple riding, and line crossings, generating immediate edge-based warnings.

3. Automated Accident Triage

Utilizes a 6-Parameter Multi-Criteria algorithm scoring emergencies based on: Vehicle Vulnerability, Location Risk, Time-of-Day, Holiday Surges, Severity Grade, and Distance Urgency. Includes a critical Fire Override feature.

4. Distributed Driver Monitoring

A Flutter mobile app streams live frames to a FastAPI backend. We utilize YOLOv11 to continuously detect drowsiness, yawning, distraction, phone use, and seatbelt compliance to alert drivers immediately.
